# What is Siarch?
It's my personal Arch installer/manager: Installing Arch using AUI script and
managing dotfiles using stow.
## Prerequisites
- Arch ISO
- Internet connection
## Download
Install git:

```
pacman -Sy git
```
Clone the script:

```
git clone https://github.com/mrkatebzadeh/Siarch.git
```
## Usage
### Installing Siarch

```
cd Siarch
./rebuild.sh
```
